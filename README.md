THE VOID
Luxury streetwear e-commerce front-end built with pure HTML, CSS, and JavaScript.
No frameworks. No backend. Just raw browser capabilities pushed properly.
What it does
Multi-page shopping experience (Home, Shop, Product, Cart, etc.)
Dynamic product rendering via JavaScript
URL-based product routing (product.html?id=N)
Fully functional cart with localStorage persistence
Scroll-triggered animations using IntersectionObserver
Responsive design with a consistent CSS design system
Stack
HTML5 (semantic structure)
CSS3 (Grid, Flexbox, custom properties)
Vanilla JavaScript (ES6+)
Key Features
Single dynamic product page serving all products
Cart state persists across pages without backend
Clean design system using CSS variables
Image hover transitions + microinteractions
No external libraries or frameworks
Why this exists
Most student projects are templates with zero depth.
This one forces real-world thinking:
state management
dynamic rendering
UX flow
scalable structure
Limitations
No backend / authentication
No real payments
Filtering is UI-only (not functional yet)
