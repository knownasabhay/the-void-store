THE VOID

Luxury streetwear e-commerce — built with raw web fundamentals.

⚡ Overview

A fully functional front-end e-commerce experience built without frameworks,libraries, or backend.

Everything — from product rendering to cart state — runs purely in the browser.

🧠 Core Idea

Most “projects” are static templates.
This isn’t.

Dynamic UI from data
Stateful cart without backend
URL-driven product pages
Real UX patterns used in production

🛠️ Tech Stack

HTML5 — semantic structure
CSS3 — Grid, Flexbox, design system
JavaScript (ES6+) — logic, rendering, state

🚀 Features

🛍️ Multi-page shopping flow (Home → Shop → Product → Cart)

🔁 Dynamic product rendering from JS data

🔗 URL-based routing 
🧾 Cart system with localStorage persistence

🎯 Scroll animations via IntersectionObserver

📱 Fully responsive layout
🎨 Centralized CSS design system
⚙️ Architecture Highlights

Single product.html serves all products
One global styles.css (design tokens + layout)
One script.js handling all logic
No duplicated markup, no hardcoded product HTML

❌ Limitations

No backend / database
No authentication system
No real checkout/payment
Filters are UI-only (not wired yet)
